**ShareCart is a real‑time, collaborative shopping list app built with .NET MAUI and Firebase.
It allows multiple users to create, edit, and share shopping lists seamlessly across devices.**

**📝 Create & Manage Lists**
Add, edit, and delete shopping items.

Auto‑scroll and auto‑focus on newly added rows.

Prevents empty items from being saved.

**👥 Real‑Time Collaboration**
Firebase real‑time sync keeps all users updated instantly.

Multiple users can edit the same list at the same time.

Each item shows who added it.

**🔐 User Accounts**
Firebase Authentication for secure login.

User profiles loaded dynamically.

**🔄 Smart UI Behaviors**
Auto-focus new entries using a custom ShouldFocus bindable property.

Pressing Enter uploads the item and creates a new row.

Smooth MVVM architecture using CommunityToolkit.

**📤 Share Lists**
Invite other users to collaborate.

Lists update instantly for all members.

🧱 Tech Stack
ShareCart está construido con un conjunto moderno de tecnologías que permiten sincronización en tiempo real, una UI fluida y una arquitectura mantenible.

Bacckend: Firebase
Frondend: Maui
